![VideoFX](https://img.shields.io/badge/VideoFX-Video%20Editor-blue?style=flat-square)

A powerful and intuitive Python library for video editing and effects processing. VideoFX provides an easy-to-use API for applying professional-grade visual effects to video files.

## Features

### Core Functionality
- 🎬 **Video Clip Management** - Load, edit, and manipulate video clips
- ✨ **Advanced Effects** - Blur, brightness, contrast, saturation, and more
- 🎨 **Color Grading** - Professional color correction and grading tools
- 🔄 **Transformations** - Resize, rotate, flip, and crop videos
- 🎵 **Audio Integration** - Handle audio tracks seamlessly
- 💾 **Multiple Format Support** - Export to MP4, MOV, AVI, WebM, and more
- ⚡ **Real-time Preview** - Preview effects before export
- 🎯 **Batch Processing** - Process multiple videos efficiently

### Visual Effects
- **Blur Effects**: Gaussian blur, motion blur, zoom blur
- **Brightness/Contrast**: Dynamic adjustment and keyframe animation
- **Color Adjustment**: Saturation, hue shift, temperature, tint
- **Distortion**: Perspective, skew, barrel distortion
- **Transitions**: Fade, dissolve, wipe, zoom transitions
- **Overlays**: Text, images, watermarks
- **Particle Effects**: Rain, snow, sparkle, confetti
